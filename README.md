\# Expense Tracker App



A simple Android Expense Tracker application built using:



\- Kotlin

\- MVVM Architecture

\- RecyclerView

\- Room Database

\- ViewModel \& LiveData



\## Features



\- Add expenses

\- View expense list

\- Categorize expenses

\- Simple clean UI

\- MVVM Architecture implementation



\## Tech Stack



\- Kotlin

\- Android SDK

\- Room

\- Coroutines

\- ViewModel
